Quick Start Scenario 2 : Play and Learn with the C Major Scale and Chord

 

One of tonespace's uses is to get insight into what scales, keys and chords are and how they relate to eachother. For this it allows you to take a chord and visually fit it on the grid that represents a musical scale in a certain key.

 

We will explain all of this in more detail later. But these steps below will already give you a quick idea of what it can do:

 

set space to Octaves [1:12]

set key to C

set scale to Major

set ch filter to All

set chord to [3] Major

set voicing to Root position (closed)

set trigger to Mouse click triggers cell

set on click to Trigger cell

set assist to Play chord anywhere

set midi out to Built-in synth

now move the mouse around in the grid and watch how the major chord interval maps onto the C Major key/scale cells

hold down the left mouse button to play the major chord at any note cell within the grid

choose other chords and/or other keys/scales and watch/hear how they map and sound
