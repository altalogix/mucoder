Quick Start Scenario 3 : Geometric Chord Generation

 

A third, more advanced use of tonespace's is to work with advanced spaces and see how chords turn out to be surprisingly simple geometric shapes in those spaces. And inversely, how you can generate chords from geometric shapes.

 

This also will be explained further down. But following steps give you a quick taste of chord generation:

 

set space to Balzano [4:3]

set labels to Note name/scale degree

set key to C

set scale to Major

set ch filter to Up to Sevenths

set voicing to Root position (closed)

set trigger to Midi triggers hovered cell

set assist to Fit most compact shape

set midi in to Midi track

set track to Pulses 8 x 1/16.

set midi out to Built-in synth

press the transport play button

now move the mouse in the grid to the following series of cells: A3/VI - C4/I - E4/III - F4/IV - D4/II - F3/IV

set voicing to Root -24 and retry the same cell sequence
